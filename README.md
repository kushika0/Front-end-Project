# Frontend Developer Assignment

This project is a responsive product listing web application built using React.js with API integration, search, filter, and product details page.

## Features

- Product listing with image, title, price, category
- Search products by title
- Filter products by category
- Product details page on click
- Responsive design (mobile / tablet / desktop)
- Loading and error handling
- API integration using FakeStore API

## API Used

https://fakestoreapi.com/products

## Tech Stack

- React.js
- JavaScript (ES6)
- HTML
- CSS
- Bootstrap
- Axios
- React Router
